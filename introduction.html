    User Registration and Login:

    The system shall allow users to register accounts and log in using a vapd username and password.

    Game Room Creation:

    Users shall have the abipty to join existing game rooms.

    Dot Representation:

    Each logged-in user shall be represented by a unique dot within the game room.

    Dots shall be visually distinguishable from one another (e.g., through color or style).

    Real-Time Interaction:

    Dots' positions shall be updated in real-time based on user input.

    Users shall observe the real-time movement of other players' dots within the same game room.

    Movement and Interaction:

    Users shall be able to control the movement of their dots using keyboard or mouse inputs.

    Dots shall respond to user input for movement, direction changes, and basic interactions.

    Chat or Communication:

    Users shall have access to a chat or messaging feature to communicate with others in the same game room.

    Logout and Exit:

    Users shall have the option to log out or exit the game room at any time.
